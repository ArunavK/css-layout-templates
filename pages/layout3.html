Neat Sidebar

This is a fullheight sidebar based navigation layout implemented using CSS Grids. The sidebar width has been adjusted using the minmax function. The sidebar never goes below a certain width but scales on increasing window size.

Hover effects and highlighting has been added as seen commonly in such designs.

Try resizing the grid using the handle on the bottom-right corner!
