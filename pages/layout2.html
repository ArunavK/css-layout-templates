Card Format using CSS Grid and Flexbox

This Flash Card format is useful for introducing people or products. The box in between can be used to display images.

It has been made using a Grid for the container and Flexbox for the actual card. The width of the card is clamped and won't increase or decrease over certain values.

Don't forget to try out different screen sizes using the handle at the corner!
